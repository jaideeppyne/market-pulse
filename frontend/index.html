    Quick guide

    1. Hot Movers

    Buy score = next-entry rank (bases, room to run, news). Quality = full checklist. Use Early buys only to hide extended names.

    2. Whale / Politician (S+ tier)

    Gold badges = named buy in news: Kela, Kacholia, Buffett, Pelosi, FII flows, etc. Scored at S+ (6.5× weight). Filter with Whale / Politician chip.

    3. Factors column

    Full scan ~90s (factors). Price tick ~45s (day % moves). Table only re-sorts when data actually changes — not a blank refresh.

    4. Earnings tab

    Who reports in the next 7 days. “TODAY” = results today.

    5. Sectors tab

    Cyclical strip = growth / cyclical / defensive counts. Each sector shows stock count, hot & early-buy counts, rotation tag, and ranked picks (non-extended first).

    6. Live updates

    Green Live = connected. Data refreshes every ~90 seconds.

    7. My List + Alerts + S+ Radar (new)

    ★ Watch any ticker (search/Analyze or hot rows). Local history + score memory. 🔔 Bell for high buy, new S+ smart money, pre-earnings. Dedicated Radar tab for named whales/politicians/FII. / focus search • w watch • f factors • e export CSV.

    8. Thesis + Sizer + Export

    Every detail/analysis now shows auto-generated setup bullets + risks + local position sizer (account + risk %). Copy full thesis as Markdown. Hot list CSV export.
